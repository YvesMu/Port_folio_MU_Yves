## 🚀 Déploiement

Ce portfolio est optimisé pour **GitHub Pages**:

1. **Clone le repository**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   ```

2. **Push vers GitHub**
   ```bash
   git remote add origin https://github.com/yourusername/portfolio.git
   git push -u origin main
   ```

3. **Activer GitHub Pages**
   - Aller à Settings → Pages
   - Sélectionner "main branch" comme source
   - Votre site sera disponible à: `https://yourusername.github.io/portfolio`

## 🔧 Personnalisation

### Modifier les contenus
1. Ouvrir les fichiers `.html`
2. Remplacer le texte générique par vos infos personnelles
3. Ajouter vos propres projets dans `projects.html`

### Modifier les couleurs
1. Éditer `assets/css/style.css`
2. Modifier les variables CSS en haut du fichier
3. Tous les éléments se mettront à jour automatiquement

### Modifier le formulaire de contact
- Le formulaire est préconfiguré avec validation
- Pour un vrai backend, intégrer un service comme Formspree, EmailJS, etc.
